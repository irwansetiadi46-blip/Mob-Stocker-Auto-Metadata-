📖 Tentang Mob Stocker

Alat untuk kontributor microstock yang ingin menghemat waktu mengisi metadata (title, description, keywords) secara otomatis menggunakan AI, lalu embed langsung ke file (bukan sidecar).

🖼️ Format Didukung

  • JPG / PNG → XMP injected langsung ke file
  • EPS → XMP injected langsung via manipulasi teks PostScript

🚀 Cara Penggunaan

  • 1. Masukkan Gemini API key → klik SAVE
  • 2. Klik tombol 📁 Pilih File atau drag & drop area upload
  • 3. Upload file (max 10, JPG/PNG/EPS)
  • 4. Klik START GENERATE (AI auto-analyze gambar)
  • 5. Atau isi manual di form metadata
  • 6. Klik EMBED → metadata akan di-embed ke file
  • 7. File siap upload ke Shutterstock/Adobe Stock

📏 Batasan Metadata

  • Title: max 100 karakter
  • Description: max 150 karakter
  • Keywords: max 49 keywords

⚙️ Teknologi

  • Google Gemini AI Vision
  • Python Flask + Pillow
  • XMP Dublin Core Schema
  • PostScript Manipulation for EPS
